Configure a custom document using the Mindee API Builder.

Parameters

  • document_type – The “API name” field in the “Settings” page of the API Builder

  • singular_name – The name of the attribute used to retrieve a single document from the API response

  • plural_name – The name of the attribute used to retrieve multiple documents from the API response

  • account_name – Your organization’s username on the API Builder

  • api_key – Your API key for the endpoint

  • version – If set, locks the version of the model to use. If not set, use the latest version of the model.

Load a document from a base64 encoded string.

Parameters

  • input_string – Input to parse as base64 string

  • filename – The name of the file (without the path)

  • cut_pdf_mode –

    Number (between 1 and 3 incl.) of pages to reconstruct a PDF with.

    • if 2: first page, penultimate page
